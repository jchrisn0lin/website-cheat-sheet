Introduction to the 3 languages needed
to build a dynamic website

HTML

The first language needed to create a web page is HTML. HTML is short for HyperText Markup Language. This represents the skeleton of the website, also reffered as: the DOM. We use it to structure the look and order of appearance of our website's elements. Common elements used in a HTML page are text, images, input, activable buttons, links, audio and video.

CSS

The second language we use to build a web page is CSS. CSS stands for Cascade Style Sheets. This code influences the HTML page to style the elements of the web page with different colors, shapes, fontstyles and overall appearance of the web page.
The use of HTML and CSS together allows the creation of static pages for a static website. Which means, a website that doesn't require the use of data or resources coming from or being sent to an external server.

JavaScript

The third language used is JavaScript. This is a programming language widely used in the world for its versatility and simplicity as an interpreted programming language. It was orignally made for website but can today be also used for different and a wide range of purposes when building software or programs. In our case, JavaScript is used to transform a static website into a dynamic website so users can interact with it. According to the website's purpose and users' needs, external data and resources can be received or sent from one or several external servers.
